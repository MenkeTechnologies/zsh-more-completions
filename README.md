## // WHAT IS THIS

A **massive** zsh completions plugin ‚ÄĒ **14,125 commands** wired into the compsys neural net. Most were auto-generated by python scripts that jacked into `--help` output and man pages, ripping completion data straight from the source. All completions have been cleaned, verified and updated with latest command options. Architecture-prefixed completions live in the `architecture_src` directory.

```
src/              -> 9,652 main completions
man_src/          -> 3,398 completions parsed from man pages
override_src/     ->     8 completions that override defaults (prepended to fpath)
architecture_src/ -> 1,067 cross-architecture completions
```

## // JACK IN

### Zinit `[RECOMMENDED]`

> `~/.zshrc`

```sh
source "$HOME/.zinit/bin/zinit.zsh"
zinit ice lucid nocompile wait'0e' nocompletions
zinit load MenkeTechnologies/zsh-more-completions
```

> **NOTE:** The `nocompletions` ice prevents zinit from symlinking all completions into `~/.zinit/completions`. Instead the plugin directly manipulates `fpath` ‚ÄĒ this lets `override_src` completions take priority over defaults while keeping system completions untouched.

### Oh My Zsh

```sh
cd "$HOME/.oh-my-zsh/custom/plugins" && git clone https://github.com/MenkeTechnologies/zsh-more-completions.git
```

Add `zsh-more-completions` to your plugins array in `~/.zshrc`.

OMZ only adds plugin root to fpath, so completions in subdirectories won't load without intervention. You need to wire the nested source dirs into fpath **before** OMZ runs `compinit`:

```sh
# OMZ does not add nested comp dirs to fpath so do it here, assume *src has completions
for plug in ${plugins[@]}; do
    if [[ -d "$ZSH/custom/plugins/$plug" ]]; then
        # null glob - no error
        for dir in "$ZSH/custom/plugins/$plug/"*src(N); do
            if [[ -d "$dir" ]]; then
                if [[ -z ${fpath[(r)$dir]} ]];then
                    if [[ $dir = *override* ]]; then
                        fpath=($dir $fpath)
                    else
                        fpath=($fpath $dir)
                    fi
                fi
            fi
        done
    fi
done
```

### Manual Install

```sh
git clone https://github.com/MenkeTechnologies/zsh-more-completions.git
```

Copy all `_*` files from the source directories to somewhere on your `fpath`.

## // BOOT SEQUENCE

After install, initialize the completion system:

```sh
autoload -Uz compinit && compinit
```

For faster shell startup, cache the `~/.zcompdump`:

```sh
compinit -C
```

This skips the security check and loads the cached dump directly ‚ÄĒ **dramatically** faster boot times.
